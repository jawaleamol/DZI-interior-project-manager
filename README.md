# BuildFlow Interior Project Manager

This repository contains a Django project called **BuildFlow**. It provides REST API endpoints for managing interior design projects, queries, bills, and documents.

## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Apply migrations:
   ```bash
   python manage.py migrate
   ```
3. Run the development server:
   ```bash
   python manage.py runserver
   ```

The API root is available at `/api/`.
